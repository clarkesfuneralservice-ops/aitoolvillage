What is Microsoft Copilot?

Microsoft Copilot is Microsoft's AI assistant powered by GPT-4 — available free at copilot.microsoft.com and deeply integrated into Microsoft 365 products including Word, Excel, PowerPoint, Teams and Outlook. For enterprise organisations running on Microsoft infrastructure, Copilot is the most seamless AI assistant available.

Microsoft Copilot for Microsoft 365 (the paid enterprise version at $30/user/month) adds AI directly into every Office application — draft emails in Outlook, generate PowerPoint presentations from a brief description, create Excel formulas from plain English, summarise Teams meetings and analyse documents in Word. This level of integration is unmatched by any other AI tool.

The free version of Copilot at copilot.microsoft.com is powered by GPT-4 and includes image generation via DALL-E 3, web search via Bing and basic productivity features. It's built into Windows 11 and the Edge browser, making it the default AI for most Windows users.
