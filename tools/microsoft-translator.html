What is Microsoft Translator?

Microsoft Translator is Microsoft's AI-powered translation service — available as a standalone app, web tool and deeply integrated across Microsoft 365 products including Teams, Word, PowerPoint, Outlook and Edge browser. For organisations running on Microsoft infrastructure, it's the most seamless translation option available.

Its standout feature is real-time conversation translation in Microsoft Teams — meeting participants can see live captions translated into their own language during calls, making multilingual meetings significantly more accessible. Word and PowerPoint documents can be translated directly within the applications without copy-pasting to external tools, preserving all formatting automatically.

For developers and enterprise teams, Microsoft Translator is available via the Azure Cognitive Services API — the same translation engine that powers Bing and Microsoft 365. It supports 100+ languages, custom translation models trained on your industry terminology, document translation at scale and real-time speech translation. Pricing is usage-based through Azure.

Frequently Asked Questions
In Microsoft Teams meetings, you can enable live captions and then select your preferred language for the captions. Microsoft Translator automatically translates the speaker's words into your chosen language in real time, displayed at the bottom of the screen. This works for both live meetings and recorded videos. The feature is available on Teams Premium plans and some Microsoft 365 business plans.
Google Translate wins on language coverage, free access limits, image translation and general consumer usability. Microsoft Translator wins for Microsoft 365 users — its Teams, Word and PowerPoint integration is significantly more seamless than using Google Translate externally. For pure translation quality, DeepL outperforms both for European languages. Choose based on your ecosystem: Google for general use, Microsoft for Office/Teams users.
The Microsoft Translator web tool and mobile apps are free to use. The Azure API for developers and enterprise use is priced per character translated — the first 2 million characters per month are free, then pricing applies based on volume. Custom translation models and document translation via the API have additional pricing. For casual personal use, Microsoft Translator is completely free.
