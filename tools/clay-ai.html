What is Clay?

Clay is a next-generation prospecting tool that uses AI to automatically research prospects from 75+ data sources and build personalised outreach at scale. Instead of manually researching each lead, Clay pulls LinkedIn data, company news, funding rounds and more — then uses AI to write personalised lines for each prospect.
